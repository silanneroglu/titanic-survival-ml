# Titanic Survival Prediction (Machine Learning)

This project predicts passenger survival on the Titanic using Logistic Regression.

Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

Project Steps
- Data loading and exploration
- Handling missing values (Age imputation)
- Encoding categorical variables (Sex → numeric)
- Train-test split (80% / 20%)
- Logistic Regression model training
- Model evaluation using accuracy score

Result
The model achieved approximately **79% accuracy** on the test set.

Purpose
This project was built to practice supervised machine learning and data preprocessing techniques.
